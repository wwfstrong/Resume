项目经历

CNode 社区

  • 项目介绍：此项目使用 Vue 还原CNode官方社区，项目调用CNode官方社区提供的 API ，以 Vue cli 创建项目模版，使用 Axios 获取数据，使用 Vue Router 进行前端路由的切换及传参，使用 watch 监听路由的变化。
  • 关键词：Vue cli/ Vue/ Vue Router/ Axios/ ES6/
  • 项目源码：github.com/Adashuai5/cnode-ada
  • 浏览链接：adashuai5.github.io/cnode-ada

网站导航

  • 项目介绍：此项目是用原生JS开发的网站导航，主体部分是仿键盘的样式，可以按键和点击访问相应的网页。该作品主要思路是，把需要创建的元素放到 Hash 中，然后通过 JavaScript 自动生成 HTML 标签和监听键盘事件的API来实现目的。
  • 关键词：原生JS/ Hash/ API/ ES6/
  • 项目源码：github.com/Adashuai5/Pikachu
  • 浏览链接：adashuai5.github.io/Pikachu

可爱皮卡丘

  • 项目介绍：此项目是使用纯原生 JS 开发，动态显示 CSS画一个皮卡丘过程的小项目。该作品主要思路是通过间隔计时器遍历预先设置在文件内的代码，将其一个个显示在页面上，并在页面样式上作同样处理，以使代码 CSS 生效。此外，还实现了动画暂停、调速、重放等功能。
  • 关键词：CSS3/ 原生JS/ ES6及模块化/ 原型对象/
  • 项目源码：github.com/Adashuai5/Pikachu
  • 浏览链接：adashuai5.github.io/Pikachu

在线画板

  • 项目介绍：此项目是使用纯原生 JS 开发的简易在线画板，主要利用 HTML5 的 Canvas API。实现了可以在PC端及手机端的在线画画，并且包含了橡皮擦、画笔调色、画笔粗细、画作删除等功能，还可以下载完成的画作。
  • 关键词：Canvas API/ 原生JS/ ES6/
  • 项目源码：github.com/Adashuai5/Canvas-ada
  • 浏览链接：adashuai5.github.io/Canvas-ada
